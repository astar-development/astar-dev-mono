## Phase 3 — Services

### `Services/ScanProgressUpdate.cs`
`record ScanProgressUpdate` with:
- `string CurrentFolder` — directory currently being walked
- `int TotalFilesProcessed` — cumulative file count
- `string? CurrentFileName` — file currently being processed (null when reporting folder change or completion)
- `string StatusMessage` — human-readable log line, e.g. `"[12:34:05] Scanning: /home/user/photos"`

### `Services/IFileTypeClassifier.cs` / `FileTypeClassifier.cs`
- Single method: `FileType Classify(string fileExtension)`.
- `fileExtension` is the value from `Path.GetExtension(filePath)` — includes the leading dot, e.g. `.jpg`.
- Implementation uses a static `Dictionary<string, FileType>` (case-insensitive) mapping extensions to
  `FileType` values. Falls back to `FileType.Unknown`.
- Extension mappings (non-exhaustive; agent should be thorough):
  - **Image**: `.jpg .jpeg .png .gif .bmp .tiff .tif .webp .svg .heic .raw .ico .avif`
  - **Document**: `.pdf .doc .docx .txt .rtf .odt .md .pages .epub`
  - **Spreadsheet**: `.xls .xlsx .csv .ods .numbers`
  - **Presentation**: `.ppt .pptx .odp .key`
  - **Video**: `.mp4 .avi .mov .mkv .wmv .flv .webm .m4v`
  - **Audio**: `.mp3 .wav .flac .aac .ogg .m4a .wma`
  - **Archive**: `.zip .rar .7z .tar .gz .bz2 .xz`
  - **Code**: `.cs .py .js .ts .java .cpp .c .h .go .rs .rb .php .html .css .json .xml .yaml .yml .sh .ps1 .sql`
  - **Database**: `.db .sqlite .sqlite3 .mdb .accdb`
  - **Executable**: `.exe .dll .so .dylib`

### `Services/IFolderPickerService.cs` / `FolderPickerService.cs`
- Interface: `Task<string?> OpenFolderPickerAsync()` — returns the selected path or `null` if cancelled.
- Implementation: uses `Application.Current.ApplicationLifetime` cast to
  `IClassicDesktopStyleApplicationLifetime` → `desktop.MainWindow` → `TopLevel.GetTopLevel(window)` →
  `StorageProvider.OpenFolderPickerAsync(new FolderPickerOpenOptions { AllowMultiple = false })`.
- Registered as `Singleton`.

### `Services/IFileScannerService.cs` / `FileScannerService.cs`
- Interface: `Task ScanAsync(string rootPath, IProgress<ScanProgressUpdate> progress, CancellationToken ct)`.
- Constructor receives `IDbContextFactory<FileAppDbContext>` and `IFileTypeClassifier`.

**Algorithm:**

1. Record `DateTime scanStartedAt = DateTime.UtcNow`.
2. Call private `Task RecurseDirectory(string dir, ...)` starting from `rootPath`.
3. `RecurseDirectory`:
   a. Report progress: `StatusMessage = $"[{time}] Scanning: {dir}"`, `CurrentFolder = dir`.
   b. For each file in `Directory.EnumerateFiles(dir)` (non-recursive, top-level only):
      - Call `ct.ThrowIfCancellationRequested()`.
      - Create `FileInfo fi`.
      - Look up existing record by `FullPath == fi.FullName` using a fresh `DbContext` from the factory.
      - If found: update `LastModified`, `SizeInBytes`, `FileType`, `LastScannedAt = UtcNow`, `PendingDelete = false`.
      - If not found: insert a new `ScannedFile` with all fields populated, `LastViewed = null`, `PendingDelete = false`, `LastScannedAt = UtcNow`.
      - Increment counter, report progress with current `CurrentFileName`.
   c. After processing all files in `dir`: `await db.SaveChangesAsync(ct)`.
   d. For each subdirectory in `Directory.EnumerateDirectories(dir)`: recurse.
4. **Only if no `OperationCanceledException` is thrown** (i.e., scan ran to completion):
   Mark-missing step — using a fresh `DbContext`:
   ```
   await db.ScannedFiles
       .Where(f => f.RootPath == rootPath && f.LastScannedAt < scanStartedAt)
       .ExecuteUpdateAsync(s => s
           .SetProperty(f => f.PendingDelete, true), ct)
   ```
5. Report final: `StatusMessage = $"[{time}] Scan complete. {total} files processed."`, `CurrentFileName = null`.

**Error handling**: Let `OperationCanceledException` propagate; let the ViewModel catch it and update UI.
Wrap `DirectoryInfo` enumeration in a try/catch for `UnauthorizedAccessException` — log the message via
progress and continue.

private IServiceProvider BuildServices()
{
    var services = new ServiceCollection();

    var dbPath = Path.Combine(
        Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData),
        "AStar.Dev.File.App", "files.db");
    Directory.CreateDirectory(Path.GetDirectoryName(dbPath)!);

    services.AddDbContextFactory<FileAppDbContext>(options =>
        options.UseSqlite($"Data Source={dbPath}"));

    services.AddSingleton<IFileTypeClassifier, FileTypeClassifier>();
    services.AddSingleton<IFolderPickerService, FolderPickerService>();
    services.AddTransient<IFileScannerService, FileScannerService>();
    services.AddTransient<MainWindowViewModel>();

    return services.BuildServiceProvider();
}

public override void OnFrameworkInitializationCompleted()
{
    var provider = BuildServices();

    // Apply EF migrations on startup
    using var ctx = provider.GetRequiredService<IDbContextFactory<FileAppDbContext>>()
        .CreateDbContext();
    ctx.Database.Migrate();

    if (ApplicationLifetime is IClassicDesktopStyleApplicationLifetime desktop)
    {
        desktop.MainWindow = new MainWindow
        {
            DataContext = provider.GetRequiredService<MainWindowViewModel>()
        };
    }
    base.OnFrameworkInitializationCompleted();
}
